#  IPL Data Analysis (2008 - 2024)

## Overview
An end-to-end Exploratory Data Analysis on the Indian Premier League (IPL) dataset 
covering 16 seasons, 1090 matches and 260,920 ball-by-ball deliveries. This project 
uncovers patterns and insights on team dominance, batting legends, bowling records 
and seasonal trends using Python.

## Key Questions Explored
- Which teams have dominated the IPL across all seasons?
- Does winning the toss actually help win the match?
- Who are the greatest batsmen and bowlers in IPL history?
- How has the IPL evolved season by season?
- Which venues have hosted the most matches?

## Key Findings
- **Mumbai Indians** lead all-time wins with 145+ victories
- **Toss winning teams won only ~51% of matches** — toss advantage is largely a myth
- **Virat Kohli** is the greatest IPL batter with 8000+ runs
- **Yuzvendra Chahal** leads wicket takers with 210+ wickets
- **IPL 2013** was the biggest season — 2014 drop due to spot fixing scandal
- **Eden Gardens** is the most used venue in IPL history

## Dataset
- Source: [IPL Complete Dataset — Kaggle](https://www.kaggle.com/datasets/patrickb1912/ipl-complete-dataset-20082020)
- matches.csv — 1090 matches, 20 features
- deliveries.csv — 260,920 deliveries, 17 features
